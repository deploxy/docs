# Available Regions

Deploxy deploys your MCP server to all of the following AWS regions simultaneously, ensuring low latency for your users around the globe.

When using the `--region` flag, you can use any of the following identifiers.

## Americas
- `us-east-1` (N. Virginia)
- `us-east-2` (Ohio)
- `us-west-1` (N. California)
- `us-west-2` (Oregon)
- `ca-central-1` (Central Canada)
- `sa-east-1` (SÃ£o Paulo)

## Europe, Middle East, and Africa
- `eu-west-1` (Ireland)
- `eu-west-2` (London)
- `eu-central-1` (Frankfurt)
- `eu-north-1` (Stockholm)
- `me-south-1` (Bahrain)
- `af-south-1` (Cape Town)

## Asia Pacific
- `ap-northeast-1` (Tokyo)
- `ap-northeast-2` (Seoul)
- `ap-northeast-3` (Osaka)
- `ap-southeast-1` (Singapore)
- `ap-southeast-2` (Sydney)
- `ap-south-1` (Mumbai)
