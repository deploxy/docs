# CLI Commands

The Deploxy CLI is the primary tool for managing and deploying your MCP server projects.

## Installation

<CodeBlock title="Terminal" language="bash">
```bash
# Global installation (recommended)
npm install -g @deploxy/cli

# Or use with npx for single commands
npx @deploxy/cli <command>
```
</CodeBlock>

## Core Commands

### `init`

Initializes a new Deploxy project in the current directory by creating a `.deploxy.json` configuration file.

<CodeBlock title="Terminal" language="bash">
```bash
npx @deploxy/cli init
```
</CodeBlock>

This command will guide you through the initial setup process.

### `deploy`

The central command for deploying your project. It bundles your code, uploads it to Deploxy, and manages the build and deployment process.

<CodeBlock title="Terminal" language="bash">
```bash
npx @deploxy/cli deploy
```
</CodeBlock>

The `deploy` command performs the following steps:
1.  **Validation**: Checks your `.deploxy.json` and `package.json`.
2.  **NPM Check**: Verifies the package version and your publish permissions.
3.  **File Collection**: Gathers all necessary source and configuration files.
4.  **Upload**: Securely uploads a compressed archive of your project to Deploxy.
5.  **Deployment**: Triggers the build, deployment, and proxy generation process on the Deploxy platform.

#### Options

-   `--config <path>`: Specify a path to a custom configuration file (e.g., `.deploxy.prod.json`).
-   `--dry-run`: Run the validation and file collection steps without uploading or deploying.

## Global Options

These options can be used with any command:

-   `--help`: Show help information for a command.
-   `--version`: Display the installed CLI version.
-   `--verbose`: Enable verbose output for debugging.
