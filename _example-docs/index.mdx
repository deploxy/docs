## Quick Start

Ready to get started? A few commands are all it takes.

<CommandPalette title="Deploy in Minutes">
  <div className="text-green-400"># Install the Deploxy CLI</div>
  <div>npm install -g @deploxy/cli</div>
  <div className="text-green-400 mt-4"># Initialize your project</div>
  <div>npx @deploxy/cli init</div>
  <div className="text-green-400 mt-4"># Deploy your server</div>
  <div>npx @deploxy/cli deploy</div>
</CommandPalette>
